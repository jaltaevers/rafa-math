RAFA'S BRICK MATH — SETUP INSTRUCTIONS
=======================================

STEP 1: Upload to Netlify (free, 30 seconds)
---------------------------------------------
1. Go to: https://app.netlify.com/drop  (on your computer)
2. Drag the ENTIRE FOLDER (all 5 files) onto the page
3. You get a URL like: https://abc-123.netlify.app
4. Send that URL to any iPhone/iPad

FILES IN THIS FOLDER:
  rafa-math-game.html   ← the game
  manifest.json         ← makes it installable
  sw.js                 ← makes it work offline
  icon-192.png          ← app icon
  icon-512.png          ← app icon (large)

STEP 2: Install on iPad as an app
----------------------------------
1. Open the Netlify URL in Safari on the iPad
2. Tap the Share button (box with arrow)
3. Tap "Add to Home Screen"
4. Tap "Add"
→ Rafa now has a Brick Math app icon on his home screen
→ It works fully offline after the first load
→ No browser address bar — looks like a real app!

STEP 3: Share with others
--------------------------
Just send the Netlify URL — works on any device, any browser.
